## Notes
- App uses Replit Auth for authentication
- Dark mode is the default aesthetic (crypto/data terminal vibe)
- Wallet connection is simulated for MVP (updates user record via API)
- Backend provides real-time-like data via standard REST endpoints (trends, prices)
